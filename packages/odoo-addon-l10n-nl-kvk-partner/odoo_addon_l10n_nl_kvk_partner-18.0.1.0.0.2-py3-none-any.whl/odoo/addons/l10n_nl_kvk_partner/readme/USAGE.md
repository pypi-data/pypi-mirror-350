To use this module, you can enter the KvK number on the partner form. It will
be stored as the Peppol Endpoint for the partner with the matching Peppol EAS
code.

![image](../static/img/l10n_nl_kvk_partner.png)

Alternatively, if the eInvoicing Endpoint for the partner is already filled in
with the matching Peppol EAS, the KvK number is visible on the partner form.
